# Research Process Log

This skill maintains process-log.md.

The process log records the visible development history of the research project.

## Purpose

Help the user show a legitimate research process.

## File

process-log.md

## Record Events

Record:

- user provided topic
- user provided questionnaire
- user provided survey data
- model was proposed
- scales were created
- Chapter 1 was written
- Chapter 2 was written
- Chapter 3 was written
- evidence_pack.md was created
- Chapter 4 was written from evidence_pack.md
- Chapter 5 was written from evidence_pack.md
- final QA was completed

## Example Entry

## 2026-07-04

User provided research topic.

Pshare classified input as TOPIC_ONLY.

Pshare created proposed research model, hypotheses, measurement scales, and questionnaire draft.

No survey results were written because no survey data was provided.

## Rules

- Do not fake user actions.
- Do not claim survey data was provided if it was not.
- Do not claim analysis was run if it was not.
