# Survey Analyzer

This module analyzes real survey data only.

## Before Analysis

Confirm:

- source file exists
- number of rows and columns
- variable names and coding
- missing values
- duplicate or invalid responses
- reverse-coded items, if any
- inclusion/exclusion criteria

## Required Evidence Pack

Create `analysis/evidence_pack.md` before writing findings. It must include:

- data source and timestamp
- sample size before and after cleaning
- cleaning rules
- descriptive statistics
- reliability results
- EFA/results if used
- correlation/regression results if used
- hypothesis decision table
- limitations

## Writing Rule

No number may appear in Chapter 4 or Chapter 5 unless it exists in `analysis/evidence_pack.md`.
