# Research Process Log

This module maintains `process-log.md`.

## Record Events

Record only real actions:

- user provided topic
- user provided questionnaire
- user provided survey data
- sources were added or verified
- model/scales were drafted
- questionnaire was revised
- evidence pack was created
- Chapter 4/5 were written from evidence
- formatting QA was completed

## Rules

- Do not fake user actions.
- Do not claim survey data was provided if it was not.
- Do not claim analysis was run if it was not.
- Do not claim final QA passed if render/inspection was not performed.
