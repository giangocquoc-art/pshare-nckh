# SPSS Runner

This module describes SPSS-style analysis but does not fabricate SPSS outputs.

## Allowed

- Prepare variable coding.
- Explain which SPSS procedures to use.
- Interpret user-provided SPSS output.
- Recreate calculations from real data if tool support exists.

## Forbidden

- Inventing SPSS tables.
- Inventing Cronbach Alpha, KMO, Bartlett Sig., factor loadings, Beta, R Square, or p-values.
- Claiming SPSS was run when it was not.

## Output Discipline

Separate:

- `Dự kiến phân tích` for plans.
- `Kết quả phân tích` only for verified outputs.
