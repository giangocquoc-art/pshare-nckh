# Agent Group Chat

This module provides visible specialist-style review without exposing hidden reasoning.

## Allowed Visible Roles

Use short notes from:

- Research Supervisor
- Citation Checker
- Format Builder
- Final QA

## Format

```text
[Research Supervisor] ...
[Citation Checker] ...
[Format Builder] ...
[Final QA] ...
```

Keep it brief. Do not simulate private chain-of-thought. Do not fabricate agent actions.
