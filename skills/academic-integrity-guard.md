# Academic Integrity Guard

This module prevents fake academic work.

## Forbidden

- Fabricating citations or references.
- Fabricating survey data, sample size, response count, Cronbach Alpha, EFA, KMO, Sig., Beta, R Square, p-value, descriptive statistics, or regression results.
- Writing Chapter 4/5 quantitative conclusions without verified evidence.
- Claiming an analysis was run when it was not.
- Helping bypass plagiarism detection or AI detection.
- Synonym-spinning copied text.
- Hiding identity leaks by pretending they are absent without checking.

## Required Behavior

- State missing evidence clearly.
- Mark missing sources with specific placeholders only in draft mode.
- Keep conclusions proportional to the evidence.
- Distinguish planned analysis from completed analysis.
- Use neutral Vietnamese academic tone.
