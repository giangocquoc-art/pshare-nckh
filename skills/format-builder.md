# Format Builder

This module prepares Euréka Word-ready research content.

## Word Baseline

- A4.
- Times New Roman 13 pt.
- Line spacing 1.3 by default, acceptable 1.3–1.5.
- Margins: left 3 cm; top 2 cm; bottom 2 cm; right 2 cm.
- Page numbers centered at top.
- Body: justified, first-line indent about 1.0–1.27 cm.
- Headings: consistent numbering.

## Structure Builder

Default order:

1. Trang bìa
2. Mục lục
3. Danh mục bảng biểu
4. Danh mục hình/sơ đồ/biểu đồ, if any
5. Danh mục từ viết tắt, if needed
6. Tóm tắt công trình, max 1 A4 page
7. Đặt vấn đề / Mô tả nghiên cứu
8. Tổng quan tài liệu / Cơ sở lý luận
9. Mục tiêu, đối tượng, phạm vi, phương pháp nghiên cứu
10. Kết quả nghiên cứu và thảo luận
11. Kết luận và kiến nghị
12. Tài liệu tham khảo
13. Phụ lục

Do not include `Lời cảm ơn` by default.

## TOC and Lists

- Do not leave `Cập nhật sau khi mở Word` in final output.
- If automatic fields cannot be updated, provide a static cleaned table or explicitly deliver draft mode only.
- Do not create empty tables as placeholders for TOC/list spacing.

## Tables

- Captions above tables.
- Source/note below tables.
- No empty giant grids.
- No detached captions.
- Fit within margins.

## Figure / Scheme / Chart Lists

- Create a list only if there are actual figures/schemes/charts.
- Do not create an empty `DANH MỤC HÌNH`.
