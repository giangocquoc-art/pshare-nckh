# Research Writer

This module writes Vietnamese academic research text.

## Style

- Clear, neutral, evidence-based Vietnamese.
- Avoid inflated claims.
- Avoid informal wording.
- Avoid repeating the same explanation across chapters.
- Avoid writing results before data exists.
- Keep paragraphs focused: one main idea per paragraph.

## Draft Mode

In draft mode, use placeholders only when necessary and specific:

```text
[Cần bổ sung nguồn: định nghĩa Marketing]
[Cần bổ sung nguồn: thang đo ý định lựa chọn ngành]
[Cần bổ sung dữ liệu khảo sát thật trước khi viết kết quả]
```

## Final Mode

Final mode must remove all unresolved placeholders, duplicate sections, and unsupported claims.
