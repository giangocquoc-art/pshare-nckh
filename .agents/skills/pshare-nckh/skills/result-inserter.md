# Result Inserter

This skill inserts verified survey results into Chapter 4 and Chapter 5.

It may only use numbers from:

```text
analysis/evidence_pack.md
```

## Responsibilities

Write:

- sample description
- descriptive statistics
- reliability results
- Cronbach Alpha interpretation
- EFA interpretation
- correlation interpretation
- regression interpretation
- hypothesis testing table
- Chapter 5 discussion
- recommendations based on actual findings

## Required Tables

When evidence is available, create:

- Bảng 4.1 Thống kê mô tả mẫu nghiên cứu
- Bảng 4.2 Kết quả kiểm định Cronbach Alpha
- Bảng 4.3 Kết quả EFA
- Bảng 4.4 Ma trận tương quan
- Bảng 4.5 Kết quả hồi quy
- Bảng 4.6 Tổng hợp kiểm định giả thuyết

## Rules

- Do not add numbers not present in `evidence_pack.md`.
- Do not change values.
- Do not overinterpret weak evidence.
- Do not claim strongest factor unless Beta values support it.
- Do not claim hypothesis accepted unless Sig. and decision rule support it.
- If evidence is incomplete, state the limitation.
