# Format Builder

This skill prepares Word-ready research content.

## Responsibilities

- assemble chapters
- create title page placeholder
- create table of contents placeholder
- format headings
- format tables
- format figures
- add questionnaire appendix
- add evidence appendix when needed
- prepare final `.docx` if tool support exists

## Recommended Structure

- Trang bìa
- Lời cảm ơn
- Tóm tắt
- Mục lục
- Danh mục bảng
- Danh mục hình
- Chương 1: Mở đầu
- Chương 2: Cơ sở lý thuyết và mô hình nghiên cứu
- Chương 3: Phương pháp nghiên cứu
- Chương 4: Kết quả nghiên cứu
- Chương 5: Kết luận và kiến nghị
- Tài liệu tham khảo
- Phụ lục

## Workflow-Specific Output

TOPIC_ONLY:

- proposal and questionnaire only

TOPIC_WITH_QUESTIONNAIRE:

- draft paper before survey
- locked Chapter 4 and Chapter 5 quantitative sections

TOPIC_WITH_SURVEY_RESULTS:

- complete paper
- Chapter 4 and Chapter 5 filled from `evidence_pack.md`
