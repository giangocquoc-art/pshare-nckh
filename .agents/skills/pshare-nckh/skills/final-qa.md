# Final QA

This skill performs final quality assurance.

Check correct workflow, no fake citations, no fake survey statistics, evidence pack exists for quantitative results, Chapter 5 conclusions are supported, numbers match evidence pack, missing sources are marked, missing data is clear, and final file is Word-ready.

Fail QA if fabricated statistics, unsupported Chapter 4/5, invented citations, invented sample size, or workflow mismatch is detected.
